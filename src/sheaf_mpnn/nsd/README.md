# 宖諺 Neural Sheaf Diffusion (NSD) Layers

This package implements **Neural Sheaf Diffusion**, a powerful Geometric Deep Learning
variant that generalizes standard Graph Convolutional Networks (GCNs) by assigning
vector spaces (**stalks**) and linear maps (**restrictions**) to the nodes and edges of
a graph.

## 宖ｧｬ Core Concept

In NSD, the graph is treated as a **Discrete Sheaf**. Instead of a single vector per
node, each node $v$ has a stalk $¥mathbf{X}_v ¥in ¥mathbb{R}^{d ¥times f}$, where $d$ is
the stalk dimension and $f$ is the hidden feature dimension.

### The Diffusion Equation

The layers follow a standardized $¥tanh$-gated diffusion process designed for numerical
stability and deep stacking:

$$¥mathbf{X}^{(t+1)} = ¥mathbf{X}^{(t)} - ¥alpha ¥cdot ¥tanh¥left(¥Delta_{¥mathcal{F}^{(t)}} ¥left(¥mathbf{I}_{nd} ¥otimes ¥mathbf{W}_1^{(t)}¥right) ¥mathbf{X}^{(t)} ¥mathbf{W}_2^{(t)}¥right)$$

Where:

* **$¥mathbf{W}_1, ¥mathbf{W}_2$**: Bilateral transform weights that mix stalk
  dimensions and features.
* **$¥Delta_{¥mathcal{F}}$**: The **Sheaf Laplacian** ($¥Delta = D - A$), dynamically
  generated by a Map Generator MLP.
* **$¥alpha$**: A learnable or fixed step size (the "graph learning rate").

## 宖屏 Model Variants

We provide three distinct sheaf variants via the `NSDVariant` Enum. Each offers a
different trade-off between expressivity and constraint:

| Variant      | Restriction Map Logic                                        | Primary Use Case                                                   |
|:-------------|:-------------------------------------------------------------|:-------------------------------------------------------------------|
| `DIAGONAL`   | Maps $¥phi_{uv}$ are diagonal matrices (vectors).            | Lightweight, computationally efficient; good for simple homophily. |
| `GENERAL`    | Maps $¥phi_{uv}$ are full $d ¥times d$ matrices.             | Maximum expressivity for complex, multi-modal relationships.       |
| `ORTHOGONAL` | Maps are constrained via **Cayley Transform** ($W^T W = I$). | Energy preservation, numerical stability in very deep networks.    |

## 宖盗 Mathematical Background

### 1. The Sheaf Laplacian ($¥Delta$)

Unlike the standard Graph Laplacian, the Sheaf Laplacian operates on the augmented
space $¥mathbb{R}^{nd}$. It is defined
as $¥Delta = ¥mathbf{D}_{¥mathcal{F}} - ¥mathbf{A}_{¥mathcal{F}}$, where:

* **$¥mathbf{D}_{¥mathcal{F}}$**: A block-diagonal matrix where each block
  is $¥sum_{u ¥in ¥mathcal{N}(v)} ¥phi_{vu}^T ¥phi_{vu}$.
* **$¥mathbf{A}_{¥mathcal{F}}$**: A block-adjacency matrix where each non-zero block
  is $¥phi_{vu}^T ¥phi_{uv}$.

When the restriction maps $¥phi$ are identity matrices, this reduces exactly to the
standard Graph Laplacian.

### 2. Bilateral Stalk Transformation

To ensure the model can learn complex feature interactions within the stalk, we apply
transformations on both sides of the stalk
matrix $¥mathbf{X} ¥in ¥mathbb{R}^{d ¥times f}$:

* **$¥mathbf{W}_1 ¥in ¥mathbb{R}^{d ¥times d}$**: Mixes the $d$ stalk dimensions (
  left-multiplication).
* **$¥mathbf{W}_2 ¥in ¥mathbb{R}^{f ¥times f}$**: Mixes the $f$ features (
  right-multiplication).

## 宖噫 Quick Start

```python
from sheaf_mpnn.nsd import NSDModel, NSDVariant

model = NSDModel(
    in_channels=128,   # raw input node features
    out_channels=7,    # target classes (e.g., Cora)
    stalk_dim=4,       # stalk dimension d
    hidden_dim=16,     # features per stalk channel f
    num_layers=4,
    variant=NSDVariant.ORTHOGONAL,
    alpha=0.01,
)
# x: [N, in_channels], edge_index: [2, E]
logits = model(x, edge_index)  # [N, out_channels]
```

## 宖ｧｪ Mathematical Properties & Testing

The implementation is verified by a rigorous, deterministic test suite:

* **Stationarity**: Verified that isolated nodes (no neighbors) result in a zero
  Laplacian, keeping the node state unchanged ($¥mathbf{X}^{(t+1)} = ¥mathbf{X}^{(t)}$).
* **Bilateral Mixing**: Confirmed that $¥mathbf{W}_1$ and $¥mathbf{W}_2$ correctly
  propagate information across stalk dimensions and features.
* **Orthogonality**: Verified that the Orthogonal variant maintains the Stiefel manifold
  property ($¥mathbf{W}^T ¥mathbf{W} = ¥mathbf{I}$) within $10^{-5}$ tolerance.
* **Permutation Invariance**: Proved that node reordering does not change the output
  logits.
